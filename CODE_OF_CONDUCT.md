# Code of Conduct

We want this project to be useful, serious, and welcoming.

## Expected Behavior

- Be respectful and direct.
- Assume good intent, but value evidence over volume.
- Keep feedback specific and actionable.
- Avoid harassment, personal attacks, doxxing, and discriminatory language.
- Respect privacy and do not share credentials, private logs, or user data.

## Unacceptable Behavior

Unacceptable behavior includes harassment, threats, sexualized attention, public posting of private information, repeated disruption, or attempts to misuse the project to harm people or systems.

## Enforcement

Maintainers may remove comments, close issues, reject contributions, or block participants who violate this code of conduct.

Security issues should be reported privately as described in [SECURITY.md](SECURITY.md).
